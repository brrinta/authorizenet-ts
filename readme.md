# authorizenet-ts


This package contains type definitions for authorizenet node-sdk
(https://github.com/AuthorizeNet/sdk-node)

#Installation

Install the package with:
> npm install -D @brrinta/authorizenet-ts 

or

> yarn add -D @brrinta/authorizenet-ts
